# Personal Portfolio - Feras Qaq

Modern, responsive portfolio website built with Flutter.

**Live Demo:** [ferasqaq-dev.github.io/portfolio_app](https://ferasqaq-dev.github.io/portfolio_app/)

## Features

- 🌓 Dark/Light Mode
- 🌐 English/Arabic Support (RTL)
- 🎨 Purple & Blue Gradient Theme
- ✨ Animated Particle Background
- 📱 Fully Responsive Design
- 🎯 Smooth Animations

## Getting Started

### Prerequisites

- Flutter SDK
- Dart SDK

### Installation

1. Clone the repository

```bash
git clone https://github.com/ferasqaq-dev/portfolio_app.git
cd portfolio_app
```

2. Install dependencies

```bash
flutter pub get
```

3. Run the app

```bash
flutter run
```

## Build for Web

```bash
flutter build web
```
